📷 Hướng dẫn Image Link

🎯 Có 2 cách để thêm ảnh sản phẩm:

1️⃣ Nhập link ảnh có sẵn

Nếu bạn đã có link ảnh từ nguồn khác (ví dụ: CDN, website khác), chỉ cần dán link vào ô Image link.

Ví dụ: https://example.com/image.jpg
2️⃣ Upload ảnh lên GitHub

Bấm nút 📷 Upload ảnh để tải ảnh từ máy tính lên GitHub. Link sẽ tự động được điền vào ô Image link.

⚠️ Lưu ý: Để sử dụng tính năng upload, bạn cần cấu hình GitHub trong Settings trước.

⚙️ Cấu hình GitHub trong Settings:

  1. Vào menu Settings (Cài đặt)
  2. Điền các thông tin sau:
    • GitHub Owner: Tên tài khoản GitHub của bạn
    • GitHub Repo: Tên repository để lưu ảnh (ví dụ: product-images)
    • GitHub Branch: Nhánh repository (thường là main)
    • GitHub Token: Personal Access Token từ GitHub
  3. Bấm Lưu cài đặt

🔑 Tạo GitHub Token:

  1. Vào GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Bấm Generate new token (classic)
  3. Chọn các quyền (scopes):
    • ✅ repo - Full control of private repositories (hoặc public_repo nếu dùng public repo)
  4. Bấm Generate token và copy token
  5. Dán token vào ô GitHub Token trong Settings

💡 Mẹo:

  • Kích thước ảnh tối đa: 5MB
  • Định dạng hỗ trợ: JPG, PNG, GIF, WebP
  • Sau khi upload, ảnh sẽ được lưu trên GitHub và hiển thị qua CDN jsDelivr
  • Bạn có thể xem preview ảnh ngay sau khi nhập link hoặc upload
